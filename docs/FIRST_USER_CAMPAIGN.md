## Builder Tester Ask

Goal: get real builders to try the extension for one AI coding session.

Message:

```text
I’m testing BuilderPerks: a tiny Chrome extension that shows useful devtool perks during Claude/ChatGPT wait time instead of dead spinner space.

Can you try it for one AI work session and tell me if it feels useful, neutral, or annoying?

Live beta + ZIP: https://builderperks.netlify.app/
```

Follow-up if they agree:

```text
The only thing I need back is blunt feedback:
1. Did you notice it?
2. Was it useful or annoying?
3. What offer would make it worth keeping installed?
```
