# GitHub Launch Funnel

## Goal

Use GitHub as the public tester and advertiser funnel for BuilderPerks.

## Primary CTA

Try BuilderPerks for one AI coding work session:

1. Download the beta extension ZIP from the latest GitHub release or from https://builderperks.netlify.app/.
2. Load it as an unpacked Chrome extension.
3. Work normally in Claude, ChatGPT, Cursor, v0, Lovable, or Bolt.
4. Leave a `Builder beta feedback` issue with one honest reaction: useful, neutral, or annoying.

https://builderperks.netlify.app/
